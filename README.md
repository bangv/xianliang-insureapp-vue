# 命令
安装依赖包：yarn add 包名(局部安装)  yarn add global 包名（全局安装）

## 客户端
开发环境（client）：yarn serve  -> 启动开发模式

测试环境（client）：yarn build:test -> 构建测试环境包   

生产环境（client）：yarn build:prod  -> 构建生产环境包

## 服务端：

启动服务进程: yarn start

查看服务端进程：pm2 ls

删除所有进程： pm2 delete all
